# 📱 VPN Client Setup

This step connects your devices to the router VPN.

---

# 🧭 Step 1: Install VPN App

Install:

* OpenVPN app OR
* WireGuard app

---

# 🧩 Step 2: Import Profile

Import VPN configuration from your router.

---

# 🔗 Step 3: Connect

Activate VPN connection and verify:

* External IP changes
* Internet still works

---

# 🎉 Result

Your device is now connected securely to your home network.
