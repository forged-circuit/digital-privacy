# 🌐 Router VPN Setup

This step enables VPN directly on your existing home router.

---

# 🧭 Step 1: Check Router Support

Log into your router and verify it supports:

* OpenVPN OR WireGuard
* VPN client or server mode

If not supported, skip to Protectli VPN.

---

# ⚙️ Step 2: Enable VPN Feature

Navigate to:

👉 Advanced Settings → VPN

Enable:

* VPN Server (or Client depending on router)
* Select protocol (WireGuard preferred if available)

---

# 🧩 Step 3: Configure VPN Network

Set:

* VPN subnet (auto or default is fine)
* Authentication method (username/password or keys)

---

# 🔐 Step 4: Save Configuration

Apply and reboot router if required.

---

# 🎉 Result

Your router now supports VPN connectivity.
