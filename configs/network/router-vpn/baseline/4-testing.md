# 🧪 VPN Testing

This step verifies your VPN is working correctly.

---

# 🧭 Step 1: External Check

Check IP address before and after VPN:

* Without VPN → normal ISP IP
* With VPN → routed IP

---

# 🧩 Step 2: Access Test

Try accessing:

* Home network devices
* Local services (if allowed)

---

# ⚠️ Step 3: Troubleshooting

If VPN fails:

* Check credentials
* Restart router
* Confirm VPN enabled

---

# 🎉 Result

VPN is functional and secure.
