# 🌐 VPN Server Configuration

This step creates your VPN server on Protectli.

---

# 🧭 Step 1: Create Tunnel

Set:

* VPN subnet (separate from LAN)
* Listen port (default 51820)

---

# 🔐 Step 2: Generate Keys

Generate:

* Private key (server)
* Public key (clients)

---

# 🎉 Result

VPN server is configured.
