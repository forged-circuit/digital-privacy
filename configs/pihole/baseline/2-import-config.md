# 📥 Step 1: Import Pi-hole Configuration

---

# 🧭 What You Are Doing

You will restore a pre-built Pi-hole configuration.

This saves time and avoids manual setup errors.

---

# ⚙️ Instructions

1. Access your Pi-hole admin interface
   (example: http://pi.hole/admin)

2. Navigate to:
   Settings → Teleporter

3. Upload the file:

👉 `pihole-backup.tar.gz`

4. Click Restore

5. Wait for Pi-hole to restart

---

# ✅ Expected Result

* Pi-hole reloads with pre-configured settings
* No manual configuration required
