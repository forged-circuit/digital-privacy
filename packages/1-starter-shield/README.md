# 🟢 Starter Shield

Entry-level network privacy and protection.

This package introduces network-wide ad blocking and gives you a choice between device awareness or password security.

---

# 🧭 What This Package Does

* Blocks ads and trackers across your entire network
* Protects all connected devices automatically
* Adds one core privacy layer (your choice)

---

# 🧩 What’s Included

### Core Protection

* Pi-hole (network-wide DNS filtering)

### Choose ONE:

* 🏠 Home Assistant
  Monitor and understand your devices

OR

* 🔑 Vaultwarden
  Secure password manager for your accounts

---

# 👤 Who This Is For

* First-time users
* Simple home networks
* Users who want basic privacy without complexity

---

# ⚙️ What You’ll Need

* A device to run Pi-hole
* Basic network access
* Optional: device for Home Assistant or Vaultwarden

---

# 🎯 Outcome

After setup:

* Ads are blocked on all devices
* Tracking is reduced
* You gain either:

  * device visibility OR
  * password security
