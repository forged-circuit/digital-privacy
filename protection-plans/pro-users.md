# 🔧 Pro Users – Protection Plans

This page shows all available protection packages with additional technical clarity.

You are not restricted to any option. You can choose any package based on your setup goals.

This guide assumes you are comfortable following structured technical instructions.

---

# 🧭 Choose Your Protection Package

## 🟢 Starter Shield

Lightweight network protection.

**Best for:**

* Simple setups
* Testing environments
* Basic network filtering

**Technical note:**

* Minimal rule set
* Low system overhead
* Easy rollback and modification

---

## 🔵 Privacy Plus (Recommended)

Balanced full-home protection system.

**Best for:**

* Most home networks
* Smart home environments
* Standard production setups

**Technical note:**

* Expanded filtering ruleset
* Stable default configuration
* Compatible with most routers and DNS setups

---

## 🟡 Business Essentials

Entry-level business-grade configuration usable in advanced home or small office environments.

**Best for:**

* Home offices
* Small teams
* Multi-device households with control needs

**Technical note:**

* More aggressive filtering logic
* Designed for mixed-device environments
* Better segmentation support

---

## 🟠 Pro Business Guard

Advanced monitoring and control layer.

**Best for:**

* Security-focused users
* Lab environments
* Small business infrastructure

**Technical note:**

* Enhanced visibility into traffic patterns
* Stronger filtering policies
* Designed for layered network control

---

## 🔴 Ultimate Offense

Maximum protection and detection system.

**Best for:**

* High-security environments
* Advanced users managing critical systems
* Enterprise-style deployments

**Technical note:**

* Strictest filtering ruleset
* High sensitivity detection
* Best paired with monitoring tools

---

# 🧠 Usage Guidance

You are free to choose any package regardless of experience level.

However:

* Starter Shield → simplest setup
* Privacy Plus → recommended default
* Business Essentials and above → increased complexity but higher control

---

# ⚙️ How to Proceed

1. Select a package above
2. Navigate to `/packages/`
3. Follow the installation guide in that package
4. Apply configuration step-by-step

---

# 🧭 Recommendation

If you're unsure where to start:

👉 Use **Privacy Plus**

It provides the best balance of stability, protection, and simplicity.
